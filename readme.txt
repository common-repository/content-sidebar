Plugin to display your sidebars to site contents.

== Description ==

This plugin is used for showing sidebars inside your content. Use shortcode [cs_sidebar id="sidebar_id"] to display the sidebar content. 


== Installation ==

1. Upload `Content Sidebar/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use shortcode `[cs_sidebar id="sidebar_id"]` in any place you want. Replace `sidebar_id` with your sidebar id.
